Arduino

Arduino est une plateforme open-source qui combine une carte électronique programmable et un langage basé sur le C++. Elle permet de créer facilement des projets interactifs en programmant des composants comme des LED, capteurs ou moteurs. Le code Arduino repose sur deux fonctions principales : setup() pour l'initialisation et loop() pour exécuter des actions en boucle. Simple et accessible, elle est idéale pour apprendre l'électronique et le codage.

Une multitude de possibilités :

  • Contrôle de robots mobiles
  • Arrosage intelligent, capteurs à distance...
  • Station météo, enregistreur de données

Exemple : feux de signalisation

Ce code contrôle un système de feux de signalisation avec Arduino. Il alterne les feux rouges, jaunes et verts pour deux directions, tout en gérant les LED pour piétons. Les délais simulent les phases de circulation.
